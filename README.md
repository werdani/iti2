# Footbar
Foobar is python library for dealing with word pluralization

# Installation

Use the package manager [pip]() to install foobar.

`pip install foobar`

# Usage
```
import footbar
footbar.pluralize('word') # returns
footbar.pluralize('word') # returns
footbar.pluralize('word') # returns
```
